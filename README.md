# Laravel Auth Lab

A hands-on Laravel project focused on learning **authentication from scratch**. Instead of relying on Laravel's built-in authentication scaffolding from the beginning, this project explores how authentication works internally and then gradually introduces Laravel's authentication features.

## 🎯 Goal

The goal of this project is to understand **how authentication actually works**, not just how to use it.

## 📚 Topics Covered

* Authentication vs Authorization
* HTTP Request & Response
* Stateless HTTP
* Sessions
* Cookies
* Password Hashing
* Registration
* Login
* Logout
* Protected Routes
* Middleware
* Auth Facade
* Manual Authentication
* Laravel Built-in Authentication
* Remember Me
* Password Reset
* Email Verification
* Authorization (Gates & Policies)

## 🛠️ Tech Stack

* Laravel
* PHP
* Blade
* MySQL
* Tailwind CSS

## 📖 Learning Approach

This project follows a step-by-step learning approach:

1. Understand the theory behind authentication.
2. Build authentication manually.
3. Learn how Laravel simplifies the process.
4. Explore advanced authentication features.

The focus is on understanding what happens behind the scenes rather than simply using pre-built authentication packages.
